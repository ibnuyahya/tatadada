Ketentuan & Privasi

1. Privasi Penuh: ReformatSuara dirancang menggunakan Web Audio API. Seluruh proses dekode dan enkoding audio terjadi di memori (RAM) browser perangkat Anda.

2. Tanpa Server: File audio Anda sama sekali tidak dikirim, diunggah, atau disimpan ke server manapun.

3. Hak Cipta: Anda bertanggung jawab atas file yang Anda konversi. Pastikan Anda memiliki hak yang sah atas materi audio tersebut.
