Kebijakan Sistem

1. Pemrosesan Lokal: Ekstraksi data XML dan konversi tabel CSV dilakukan sepenuhnya di memori browser Anda (Client-Side). Tidak ada data yang diunggah ke server eksternal.

2. Teknologi Heuristik: Sistem akan berusaha mendeteksi tag meta seperti <Title> atau <Author> pada dokumen XML Anda untuk memberikan nama file CSV yang logis dan rapi.

3. Regex Fallback: Apabila struktur XML terpotong, sistem akan memaksa pengambilan data mentah dari hierarki tag <Row> terdekat.
